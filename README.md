# BankShield AI

AI-Driven Intrusion Detection System for Banking IoT Networks

## Overview

BankShield AI is a cybersecurity platform designed for banking and IoT environments. It combines machine learning, explainable AI, and real-time network monitoring to detect suspicious activity and support security analysts.

## Features

* Random Forest Intrusion Detection System
* Isolation Forest Zero-Day Detection
* SHAP Explainability
* Security Copilot
* Live Network Packet Monitoring
* FastAPI Backend
* Streamlit Dashboard
* Threat Intelligence Module
* Incident Clustering
* Business Impact Analysis

## Datasets

* UNSW-NB15
* CICIDS2017

## Model Performance

* Accuracy: 96.82%
* SMOTE-Based Data Balancing
* Explainable Predictions using SHAP

## Technology Stack

* Python
* FastAPI
* Streamlit
* Scikit-Learn
* SHAP
* Scapy
* Plotly
* Pandas
